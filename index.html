What is Slopify?

Slopify is the result of the ideomotor effect applied to software: A 100% community-driven open source website. The premise? Commit whatever code you want for whatever features you want in whatever language you want, as long as everything works together and is legal.

Slopify operates through "Seasons" - with each season being a collaborative effort. You can explore the results and details of each season as follows:
